# 원티드 프리온보딩 프론트엔드 - 선발 과제

> 원티드 프리온보딩 프론트엔드 과제를 수행하고 결과를 제출하기 위한 레포지토리입니다.

## 🚀 프로젝트 실행 방법

### 1. 의존성 설치

```bash
npm install
```

### 2. 클라이언트 실행

```bash
npm start
```
